**Vercel Pro 플랜 ($20/월)**:
- 기본료 $20에 $20 크레딧 포함 (모든 인프라 사용료에 공유)
- Build Minutes: $7.50/시간 → $20 크레딧으로 약 2시간 40분 커버
- 초과분은 on-demand 과금 (주기 마감 시 청구)
- Hobby 플랜은 100시간 상한, 넘으면 빌드 멈춤 (추가 과금 없음)

**정상 사용 비용 감각**:
- 빌드 1회 ≈ 2분 ≈ $0.25
- 하루 5회 배포 ≈ $1.25
- 한 달 ≈ $37 (12개 프로젝트, 각 하루 1회 /bye 기준)

**Spend Management**: Settings → Billing → Budget $10 + Pause Production ON 설정됨 (2026-04-18)
